# Common

This directory contains shared scripts and utilities used by the reference architectures.

## Purpose

The files in this directory provide common functionality such as:

- Job execution (`run.sh`)
- Shared utilities and helpers
- Core logic reused across workflows

## Usage

These components are used internally by the reference architectures under:

```
reference_architectures/
```

Users typically do not need to modify files in this directory.

## Notes

- Changes here may affect all reference architectures
- This directory is part of the core execution framework

## Summary

- Shared infrastructure for all workloads
- Reused across configurations and workflows
- Not intended for direct standalone use
